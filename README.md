# Women Productivity Prediction using Cycle Syncing (Machine Learning)

## 📌 Project Overview

This project focuses on predicting women's productivity levels using the **cycle sync method** combined with **machine learning techniques**. The model analyzes how different phases of the menstrual cycle influence energy levels, focus, and work efficiency, and then predicts productivity levels accordingly.

The aim is to create a data-driven approach to align tasks and work patterns with natural hormonal cycles, helping improve productivity and well-being.

## 🌀 What is Cycle Syncing?

Cycle syncing is a method that aligns activities with the four phases of the menstrual cycle, as hormonal fluctuations directly affect physical and mental performance.

### Menstrual Cycle Phases

#### 1️⃣ Menstrual Phase (Days 1–5)

* Hormone levels: Low estrogen and progesterone
* Energy levels: Low
* Suitable activities: Rest, reflection, light planning

#### 2️⃣ Follicular Phase (Days 6–13)

* Hormone levels: Rising estrogen
* Energy levels: Increasing
* Suitable activities: Learning, brainstorming, initiating tasks

#### 3️⃣ Ovulatory Phase (Days 14–16)

* Hormone levels: Peak estrogen and testosterone
* Energy levels: Highest
* Suitable activities: Presentations, communication, collaboration

#### 4️⃣ Luteal Phase (Days 17–28)

* Hormone levels: High progesterone
* Energy levels: Moderate to low
* Suitable activities: Task completion, analysis, organized work

## 🧠 Machine Learning Approach

The machine learning model learns the relationship between menstrual cycle phases, lifestyle factors, and productivity levels.

 Input Features

* Menstrual cycle phase
* Hormone levels (optional)
* Sleep duration
* Stress level
* Mood
* Work hours

 Target Variable

* Productivity Level (Low / Medium / High)

 ML Workflow

1. Data collection and preprocessing
2. Feature encoding and scaling
3. Model training and evaluation
4. Productivity prediction

📊 Results & Insights

* Productivity varies significantly across menstrual cycle phases
* Ovulatory and follicular phases show higher productivity levels
* Luteal and menstrual phases benefit from lighter or structured tasks

🎯 Applications

* Workplace well-being analytics
